# prisma-postgresql-client-adapter

**THIS PROJECT IS A WORK IN PROGRESS**

## Developer 

### Setup

```bash
make install
```

### Run Tests

```bash
make test
```

### Compile

```bash
pnpm run compile
```

### Production Build (before release)

```bash
pnpm run build
```


### List available make targets

```bash
## without any args
make 
## or this is the same (help is the default target)
make help
```
